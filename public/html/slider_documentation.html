Slider Plugin Documentation

The Slider Plugin is a jQuery-based image slider that provides a customizable and feature-rich solution for creating responsive slideshows on web pages.

Installation

Include jQuery and the slider.js file in your HTML:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<script src="path/to/slider.js"></script>

Basic Usage

To create a basic slider, use the following HTML structure:

<div id="slider">
    <div class="slides">
        <div class="slide">
            <img src="image1.jpg" alt="Slide 1">
        </div>
        <div class="slide">
            <img src="image2.jpg" alt="Slide 2">
        </div>
        <div class="slide">
            <img src="image3.jpg" alt="Slide 3">
        </div>
    </div>
    <div class="pagination"></div>
    <button class="prev22">Previous</button>
    <button class="next22">Next</button>
</div>

Initialize the slider with jQuery:

$('#slider').sliderPlugin();

Advanced Usage

You can customize the slider with various options:

$('#slider').sliderPlugin({
    pagination: true,
    navigation: true,
    navigationStyle: 'icon',
    paginationStyle: 'dots',
    mouseDrag: true,
    autoSlide: true,
    loop: true,
    autoSlideTime: 3000,
    animation: 'fade',
    animationTime: 500,
    stopSlideOnHover: true
});

Options

Option Type Default Description
pagination boolean true Enable/disable pagination
navigation boolean true Enable/disable navigation buttons
mouseDrag boolean true Enable/disable mouse drag functionality
dragCursor string 'grabbing' Custom cursor style during drag
autoSlide boolean false Enable/disable auto sliding
loop boolean false Enable/disable infinite loop
autoSlideTime number 1000 Time interval for auto sliding (in milliseconds)
animation string 'slide' Animation type (slide, fade, zoom, flip, bounce, rotate, scale, blur, slide-up, slide-down)
animationTime number 1000 Duration of animation (in milliseconds)
navigationStyle string 'text' Style of navigation buttons (text, icon, text_w_b, icon_w_b, icon_f_s_h, text_f_s_h, icon_text_h, icon_text)
leftText string "Previous" Text for the left (previous) navigation button
RightText string "Next" Text for the right (next) navigation button
LeftIcon string SVG code SVG icon for the left navigation button
RightIcon string SVG code SVG icon for the right navigation button
paginationStyle string 'number' Style of pagination (number, dots, squire_dots, titles, titles_flex)
stopSlideOnHover boolean false Pause auto-sliding when hovering over the slider
paginationOutlet string 'outset' Position of pagination (inset, outset)

Methods

The plugin returns an object with the following methods:

var slider = $('#slider').sliderPlugin();

// Go to a specific slide
slider.goToSlide(2);

// Start auto sliding
slider.startAutoSlide();

// Destroy the slider
slider.destroy();

// Recreate the slider
slider.recreate();

Custom Slider Examples

1. Full-width Slider with Custom Navigation

<div id="full-width-slider" style="width: 100%;">
    <div class="slides">
        <div class="slide" style="background-image: url('image1.jpg'); background-size: cover; height: 400px;"></div>
        <div class="slide" style="background-image: url('image2.jpg'); background-size: cover; height: 400px;"></div>
        <div class="slide" style="background-image: url('image3.jpg'); background-size: cover; height: 400px;"></div>
    </div>
    <div class="pagination"></div>
    <button class="prev22"><i class="fas fa-chevron-left"></i></button>
    <button class="next22"><i class="fas fa-chevron-right"></i></button>
</div>
$('#full-width-slider').sliderPlugin({
    navigationStyle: 'icon',
    paginationStyle: 'dots',
    autoSlide: true,
    loop: true,
    animation: 'fade'
});

2. Content Slider with Titles

<div id="content-slider">
    <div class="slides">
        <div class="slide" sliderName="Slide 1">
            <h2>Slide 1 Title</h2>
            <p>Content for slide 1</p>
        </div>
        <div class="slide" sliderName="Slide 2">
            <h2>Slide 2 Title</h2>
            <p>Content for slide 2</p>
        </div>
        <div class="slide" sliderName="Slide 3">
            <h2>Slide 3 Title</h2>
            <p>Content for slide 3</p>
        </div>
    </div>
    <div class="pagination"></div>
</div>
$('#content-slider').sliderPlugin({
    navigation: false,
    paginationStyle: 'titles',
    mouseDrag: true,
    animation: 'slide'
});

Custom Updates

You can update the slider's options after initialization:

var slider = $('#slider').sliderPlugin();

// Change animation type
slider.recreate({
    animation: 'zoom'
});

// Enable auto sliding
slider.recreate({
    autoSlide: true,
    autoSlideTime: 5000
});

// Change navigation style
slider.recreate({
    navigationStyle: 'icon_text'
});
